Схема работы регистрации:
1. вводятся все данные регистрации в форму регистрации
2. далее производится перевод добровольного пожервтования
3. сайт проверяет валидность пдф чека
4. если все корректно, данные о регистрации пишутся в бд, перекачиваются в отдельную табличку на яндекс диск, пользователю приходит уведомление на электронную почту о переводе и о регистрации
5. чек сохраняется, чтобы потом его можно было бы глянуть в админке
