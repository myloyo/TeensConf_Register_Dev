Разработка сайта регистрации для конференции ТИНС

сайт захостчен на сервере, доступен тут: https://wolsar.site/

Схема работы регистрации:
1. вводятся все данные регистрации в форму регистрации
2. далее производится перевод добровольного пожервтования
3. сайт проверяет валидность пдф чека
4. если все корректно, данные о регистрации пишутся в бд, перекачиваются в отдельную табличку на яндекс диск, пользователю приходит уведомление на электронную почту о подтверждении регистрации
5. служители и прочие могут смотреть приходящие регистрации и чеки в табличке яндекс диска

используемые технологии: Docker, Spring Boot, Thymeleaf и JavaMailSender для работы с почтой, работа с апи Яндекс Диска
